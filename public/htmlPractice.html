Html Element and Attributes practice

Tag name:- <div>

Description:- It stands for "division" or "division container". It is a block-level container. it doesn't have any specific visual formatting. It is used to group together other HTML elements. Also used for styling, scripting, or layout to a specific section of content.

Important Attributes:- id, class, style, title,

div tag as a block level element

Tag name:- <p>

Description:- <p> tag stands for "paragraph". It is a block-level element that is commonly used for structuring and formatting text content. Avoid nesting block-level elements inside p tag, as this is invalid HTML and inconsistent rendering.

Important Attributes:- id, class, style, title,

p tag as a block level

Tag name:- <iframe>

Description:- <iframe> tag is used to embed another web page or content within the current web page. It stands for "inline frame" and is commonly used to display external content like videos, maps, or other websites.

Important Attributes:- src, width, height, frameborder, scrolling

Tag name:- <img>

Description:- It is used to embed images on a web page. It allows to display images such as photographs, illustrations, icons, and other graphical content.

Important Attributes:- src, alt, width, height, title, style

Tag name:- <pre>

Description:- It is used to define performatted text. performatted text is displayed in a fixed-width font and preserves both spaces and line breaks. This tag is useful for displaying text that should maintain its original formatting, such as code snippets or ASCII art.

Important Attributes:- it does not have specific attributes. it is used with other tags <code> and <samp>
